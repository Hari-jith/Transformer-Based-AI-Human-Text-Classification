# 🤖 AI vs Human Text Detection using DistilBERT

## 📌 Overview

This project focuses on detecting whether a piece of text was written by a human or generated by an Artificial Intelligence (AI) model. The system leverages **DistilBERT**, a lightweight transformer-based language model, fine-tuned for binary text classification using **PyTorch** and **Hugging Face Transformers**.

The project explores not only model performance on benchmark datasets but also investigates the challenges of real-world AI text detection through out-of-distribution evaluation.

---

## 🎯 Objectives

* Develop a binary text classification model for AI-generated text detection.
* Fine-tune a pre-trained transformer model using transfer learning.
* Evaluate model performance using industry-standard metrics.
* Analyze model behavior through confusion matrices and classification reports.
* Assess real-world generalization capabilities on unseen text samples.
* Establish a baseline for future explainability and deployment enhancements.

---

## 📚 Error Analysis

The real-world evaluation revealed several interesting patterns:

### Human → AI Misclassification

Human-written content such as:

* News articles
* Sports articles
* Wikipedia excerpts

was sometimes incorrectly classified as AI-generated.

### AI → Human Misclassification

Some AI-generated content written in:

* Narrative style
* Historical writing style
* Informative article format

was incorrectly classified as human-written.

### Insight

The findings suggest that the model partially relies on stylistic patterns present in the training data rather than fully understanding authorship characteristics.

This highlights a key challenge in AI-generated text detection:

> High benchmark accuracy does not necessarily guarantee strong real-world generalization.
