# AI vs Human Text Detection using DistilBERT

A Natural Language Processing project that uses **DistilBERT** and PyTorch to classify whether a piece of text is written by a human or generated by AI.

The model was trained using a balanced subset of the **artem9k/ai-text-detection-pile** dataset and evaluated on multiple external datasets to assess generalization ability.

---

## Overview

Large language models can generate highly fluent text that closely resembles human writing. Detecting AI-generated content has become increasingly important in domains such as:

- Education
- Journalism
- Content moderation
- Research
- Plagiarism detection

This project fine-tunes a DistilBERT transformer model for binary text classification:

- Human-written text → Label 0
- AI-generated text → Label 1

## Dataset

### Training Dataset

Dataset:

**artem9k/ai-text-detection-pile**

Original dataset size:

| Class | Samples |
|---------|---------:|
| Human | 1,028,146 |
| AI | 364,376 |

A balanced subset was created:

| Class | Samples |
|---------|---------:|
| Human | 30,000 |
| AI | 30,000 |
| Total | 60,000 |

After text cleaning and filtering:

```
Final Shape: (59,111, 4)
```

---

## Data Preprocessing

The following preprocessing steps were performed:

- Remove missing values
- Convert texts to string
- Remove empty samples
- Keep samples with more than 20 words
- Shuffle dataset
- Stratified train-validation-test split
