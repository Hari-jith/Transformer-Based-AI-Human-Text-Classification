# 🤖 AI vs Human Text Detection using DistilBERT

## 📌 Overview

This project focuses on detecting whether a piece of text was written by a human or generated by an AI model. The system leverages **DistilBERT**, a lightweight transformer-based language model, fine-tuned for binary text classification using **PyTorch** and **Hugging Face Transformers**.

The objective is to build an efficient AI-generated content detector capable of distinguishing between human-written and AI-generated text across a large-scale dataset.

---

## 🎯 Objectives

* Develop a binary text classification model for AI text detection.
* Fine-tune a pre-trained transformer model using transfer learning.
* Evaluate model performance using industry-standard classification metrics.
* Analyze model behavior through confusion matrices and classification reports.
* Establish a strong baseline for future enhancements such as explainability, confidence analysis, and deployment.

---

The following preprocessing steps were performed:

* Removed missing text entries
* Removed empty text samples

* Converted labels into numerical format
* Created balanced class distribution

## 🧠 Model Architecture

### DistilBERT

Model Used:

distilbert-base-uncased

DistilBERT is a compressed version of BERT that retains most of BERT's language understanding capability while reducing computational requirements.

### Configuration

* Maximum Sequence Length: 128
* Batch Size: 16
* Optimizer: AdamW
* Learning Rate: 2e-5
* Epochs: 3
* Loss Function: CrossEntropy Loss (handled internally by DistilBERT)

## 🔍 Key Findings

* DistilBERT achieved strong classification performance with over 95% test accuracy.
* The model demonstrated balanced performance across both classes.
* Increasing dataset size from 20,000 to 50,000 samples improved overall generalization and reduced classification errors.
* Preliminary real-world testing revealed that conversational human-written text can sometimes be misclassified as AI-generated content, highlighting the challenges of out-of-distribution generalization in AI text detection systems.
